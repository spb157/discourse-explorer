# Discourse Explorer — d+m

Interactive tool for exploring Cultural Discourse Analysis territories. Upload a discourse analysis JSON, choose a strategic lens, and navigate the 2×2 landscape map.

## Deployment: GitHub + Netlify

### 1. Push to GitHub

```bash
# In the project folder
git init
git add .
git commit -m "Discourse Explorer v1"

# Create a repo on GitHub (private recommended), then:
git remote add origin https://github.com/YOUR-ORG/discourse-explorer.git
git branch -M main
git push -u origin main
```

### 2. Deploy on Netlify

1. Go to [app.netlify.com](https://app.netlify.com) → **Add new site** → **Import an existing project**
2. Connect your GitHub account and select the `discourse-explorer` repo
3. Build settings should auto-detect from `netlify.toml`:
   - **Build command:** `npm run build`
   - **Publish directory:** `dist`
4. Click **Deploy site**

### 3. Enable AI features (optional)

The AI features (generate new lenses, refine narratives, generate provocations) require an Anthropic API key.

1. In Netlify: **Site settings** → **Environment variables**
2. Add: `ANTHROPIC_API_KEY` = your key
3. Redeploy the site

Without the key, everything works except the three AI generation features — the tool will show an error message if someone tries to use them.

### 4. Custom domain (optional)

In Netlify: **Domain management** → **Add custom domain**

## Local development

```bash
npm install
npm run dev
```

For AI features locally, create a `.env` file:
```
ANTHROPIC_API_KEY=sk-ant-...
```

Then run with Netlify CLI:
```bash
npx netlify dev
```

## Usage

1. **Upload** a discourse analysis JSON (generated by the Claude Discourse Project)
2. **Choose a lens** — select from pre-loaded axes or generate a custom one
3. **Explore** — navigate the map, tensions, provocations, narrative, and sources
